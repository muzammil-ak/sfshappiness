🍎 Save Food, Share Happiness
A full-stack solution to combat food waste by connecting donors with people in need, ensuring that no meal goes to waste.
📌 Project Overview
Save Food, Share Happiness is a social impact platform designed to reduce food waste and help people in need. This application brings together donors, volunteers, and organizations on one platform so that surplus food can reach the right place at the right time.
🌟 Key Features
Multi-Page Navigation:
Includes Home, About, Donate, and Volunteer sections for a complete user experience.
Smart Donation System:
A user-friendly "Donate Now" button that allows donors to quickly contribute.
Gmail Server Integration:
Backend is connected with a Gmail server to enable direct communication, notifications, and updates.
Awareness Hub:
A dedicated section that highlights the impact of food waste and ways to prevent it.
Terminal-Ready:
Backend is fully integrated and can be easily run on localhost using the VS Code terminal.
🚀 Tech Stack
Frontend: React.js (Component-based architecture), HTML5, CSS3, JavaScript (ES6+)
Backend: Node.js / Express.js (Server-side logic)
Communication: Gmail Server Integration for automated emails
Environment: Localhost testing via terminal
🏗️ Project Structure
The application includes the following main pages:
Home: Introduction and key highlights of the platform
About: Mission and vision of the project
How It Works: Complete donation process explained
Donate: Form to submit food donation details
Volunteer: Section to join and contribute to the community
Awareness: Informative guide to reduce food waste
Contact: Direct message portal for queries
